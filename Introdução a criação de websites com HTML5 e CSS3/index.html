Lógica de Programação com JavaScript

Ensinar lógica de programação usando a linguagem JavaScript para iniciantes na área de programação usando conceitos e exemplos do dia a dia. Ensina a base de uma linguagem de programação: comandos básicos (entrada/saída), variáveis, estruturas de decisão, estruturas de repetição, funções e vetores. O conteúdo programático inclui conhecimento do mundo web: introdução à lógica de programação, variáveis, operadores e entrada de dados, algoritmos inteligentes, otimização com dados repetidos, organização e melhoramento do código. Ao finalizar este curso, o aluno será capaz resolver problemas de programação usando raciocínio lógico, ler e entender logicamente o que o código-fonte de um programa expressa, discutir soluções de arquitetura básica e componentes de um sistema, escrever algoritmos visando a completude de uma tarefa e aprender virtualmente qualquer linguagem de programação Curso totalmente gratuito, na modalidade ensino à distância com carga horária de 20h e certificado de conclusão e para realizá-lo, é necessário apenas conhecimento básico de informática.
